Welcome!

To try out this API, make a GET request to /stats with the following query parameters:

  • career: Any string, e.g. Database Analyst (required)
  • city: Any string, e.g. Ottawa
  • province: Any string, E.g. Ontario or ON
  • country: CA or US (required)
  • experience: [empty string representing all], entry_level, mid_level, senior_level
  • position: [empty string representing all], temporary, parttime, fulltime, internship, contract, commission

Example usage:

http://api.davidliao.ca/stats?career=Database%20Analyst&city=Ottawa&province=ON&country=CA&experience=&position=
